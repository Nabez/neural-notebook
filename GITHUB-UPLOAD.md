# Neural Notebook - GitHub Upload Guide

## Schnellstart

1. Erstelle auf GitHub ein Repository: https://github.com/new
   - Name: `neural-notebook`
   - Public oder Private
   - NICHT initialisieren (kein README, .gitignore, etc.)

2. Auf deinem Mac Terminal:

```bash
# Ordner erstellen
mkdir neural-notebook
cd neural-notebook

# Git initialisieren
git init

# package.json erstellen (kopiere von unten)
# ... alle Dateien erstellen ...

# Commit & Push
git add .
git commit -m "Initial commit - Neural Notebook"
git remote add origin https://github.com/nabez/neural-notebook.git
git branch -M main
git push -u origin main
```
